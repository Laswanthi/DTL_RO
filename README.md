# DTL_RO
Dynamic Transport Logistics & Route Optimization Tool (DTL-RO) is a Python-based project used to find the shortest and best route between locations. It represents locations as nodes and roads as edges in a graph. The system uses Dijkstra’s Algorithm to calculate the minimum distance from source to destination. It helps reduce travel time, fuel consumption, and transportation cost, making it useful for delivery and logistics services.
